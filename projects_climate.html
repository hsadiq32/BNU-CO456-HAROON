a weather forecast application.

climate - made with python

The project I am undergoing is the task of creating a weather app that would still predict the weather even offline. The program should have a user friendly GUI with a way to access the GPS or map coordinates, the user should be able to enter their location and then receive the following forecast relevant to the what has been inputted to do this I need to insert an API key from an outside forecast provider to give me the data required to do the job. I would need to create a database to make a user login system where locations and other data can be saved. This will also be the place where the barometer, hydrometer and open weather map API can be stored and used to make the code more accurate by working out a mean in blocks. The next thing I plan on fixing is predicting weather on the go, what I mean by that is the app will predict the weather while considering your route, to do this you can select the average walking speed then the program will calculate if there may be potential rain/thunder on your route at that predicted time.
